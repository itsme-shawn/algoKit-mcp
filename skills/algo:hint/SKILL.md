# algo:hint - 백준 문제 힌트 생성

백준 온라인 저지(BOJ) 문제를 분석하고 단계별 힌트를 제공하는 스킬입니다.

## 사용법

```
/algo:hint <문제번호>
```

**예시:**
- `/algo:hint 1003` - 피보나치 함수 문제 분석
- `/algo:hint 11729` - 하노이 탑 이동 순서 문제 분석

## 동작 방식

1. **MCP 도구 호출**: `algokit/analyze_problem` 사용
2. **문제 분석 정보 수신**:
   - 문제 제목, 난이도(티어), 태그 정보
   - 3단계 힌트 가이드 프롬프트
   - 알고리즘별 학습 컨텍스트
3. **사용자 상황 파악**: 현재 진행 상황 분석
4. **적절한 힌트 제공**:
   - **Level 1**: 문제 접근법 (사용자가 막힌 경우)
   - **Level 2**: 구현 힌트 (부분 구현 완료)
   - **Level 3**: 상세 풀이 (명시적 요청 시)

## MCP 도구 사용

반드시 다음 순서로 실행:

```bash
# 1. 스키마 확인 (필수)
mcp-cli info algokit/analyze_problem

# 2. 문제 분석 실행
mcp-cli call algokit/analyze_problem '{"problemId": 1003}'
```

## 힌트 레벨 선택 가이드

사용자 상황을 파악하고 **단 1개 레벨만** 제공:

| 상황 | 선택 레벨 | 판단 기준 |
|------|-----------|-----------|
| 문제 처음 봄 / 막힘 | Level 1 | 코드 없음, "어떻게 시작?", "접근법?" |
| 일부 구현 완료 | Level 2 | 코드 작성 중, "더 필요해", "구현 막힘" |
| 정답 요청 | Level 3 | "정답", "풀이", "코드 보여줘" 명시 |

## 주의사항

- ❌ 3개 레벨 모두 제공 금지
- ✅ 사용자 상황 분석 후 1개만 선택
- ✅ 프롬프트 가이드를 활용해 자연스러운 힌트 생성
- ✅ 사용자가 "더 힌트" 요청 시 다음 레벨로 진행

## 관련 스킬

- `/algo:review` - 풀이 후 복습 템플릿 생성
- `/algo:search` - 유사 난이도 문제 검색
