# algo:review - 백준 풀이 복습 템플릿

백준 문제 풀이 후 복습 문서를 생성하는 스킬입니다.

## 사용법

```
/algo:review <문제번호>
```

**예시:**
- `/algo:review 1003` - 피보나치 함수 복습 문서 생성
- `/algo:review 11729` - 하노이 탑 복습 문서 생성

## 동작 방식

1. **MCP 도구 호출**: `algokit/generate_review_template` 사용
2. **템플릿 및 가이드 수신**:
   - 마크다운 복습 템플릿
   - 문제 분석 정보 (난이도, 태그, 설명)
   - 복습 가이드 프롬프트
   - 관련 문제 추천
3. **대화형 복습 작성**:
   - 사용자 코드 분석
   - 시간복잡도 계산
   - 개선 포인트 제안
   - 학습 내용 정리

## MCP 도구 사용

반드시 다음 순서로 실행:

```bash
# 1. 스키마 확인 (필수)
mcp-cli info algokit/generate_review_template

# 2. 복습 템플릿 생성
mcp-cli call algokit/generate_review_template '{"problemId": 1003}'
```

## 복습 문서 구조

생성되는 마크다운 템플릿:

```markdown
# [1003] 피보나치 함수

## 문제 요약
- **난이도**: Silver III (티어 8)
- **알고리즘**: DP, Memoization
- **풀이 날짜**: YYYY-MM-DD

## 핵심 아이디어
<!-- 사용자와 대화하며 작성 -->

## 내 풀이

### 코드
```python
# 사용자 코드 삽입
```

### 시간복잡도
<!-- 분석 결과 작성 -->

## 배운 점
<!-- 학습 내용 정리 -->

## 개선 방향
<!-- 개선 포인트 제안 -->

## 관련 문제
- [9184] 신나는 함수 실행
```

## 대화형 작성 프로세스

1. **코드 확인**: "풀이 코드를 보여주세요"
2. **분석 및 질문**:
   - "어떤 접근법을 사용하셨나요?"
   - "어려웠던 부분은 무엇인가요?"
3. **복습 문서 작성**:
   - 템플릿 기반으로 섹션별 작성
   - 가이드 프롬프트 활용
4. **저장 위치 제안**: `reviews/{문제번호}_REVIEW.md`
   - 예시: `reviews/1003_REVIEW.md`
   - 폴더 구조: `BOJ/10000~/15683/15683_REVIEW.md`

## 주의사항

- ✅ 사용자와 대화하며 단계별 작성
- ✅ 가이드 프롬프트로 자연스러운 질문 생성
- ✅ 사용자 코드 기반 맞춤형 피드백
- ❌ 단순 템플릿 덤프 금지

## 관련 스킬

- `/algo:hint` - 문제 힌트 생성
- `/algo:search` - 관련 문제 검색
